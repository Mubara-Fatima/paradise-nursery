# Paradise Nursery

Paradise Nursery is a dynamic shopping cart web application for an online plant shop. 
The app allows users to browse a variety of houseplants organized by category, view 
plant details such as images, names, descriptions, and prices, add items to a shopping 
cart, and manage cart quantities in real time.

## Features
- Browse houseplants across multiple categories
- Add products to cart with dynamic quantity and total price updates
- Responsive navigation bar (Home, Plants, Cart)
- Built with React and Redux Toolkit

## Tech Stack
- React
- Redux Toolkit
- CSS
